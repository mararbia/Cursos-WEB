Formulário

O formulário é muito utilizado para coleta de dados. As tags mais utilizadas serã descritas abaixo:

Get

Testando o get e enviando dados para a busca no google Digite na caixa o nome do item a ser pesquisado e aperte enter
O action determina para qual URL a requisão get ou post será enviada

Input










Setando o value, ele já deixa um valor para ser salvo. O ideal é deixar ele vazio
  • O text mostra o texto puro
  • O password esconde o texto
  • O email verifica se o formato é de e-mail e abre o teclado do celular com as opções de e-mail
  • O hidden esconde o campo, geralmente é utilizado com o JS
  • O search é utilizado quando o campo for de busca, ele adiciona um elemento que permite apagar o conteúdo de dentro
  • O tel permite em um dispositivo móvel que o apareça somente o teclado numérico ao digitar o número e caso de clique no número ele abre o discador
  • O number só aceita números
  • A url valida se é uma URL, ou seja, se tem o http:// ou https://
  • O range é um slider no qual podemos colocar o min, max e o salto que ele vai dar

Checkbox

É um tipo de input só que com a possibilidade de selecionar vários valores para um mesmo dado
Segunda-Feira
Quarta-Feira
Sexta-Feira

Radio Button

Permite que apenas uma opção seja escolhida Manhã Tarde Noite

Text Area

Área de digitação de um texto. O texto digitado entre as tags de abertura e fechamento vai aparecer dentro da text area

Select

Conhecido também como dropdown. Utilizando quando tem muitas opções e o usuário já conhece a resposta

Datalist

Permite começa a digitar um texto e ele identifica o restante do texto. Ele está sempre conectado a um input.

Button

  • Type button deixa ele como apenas um botão, não tem ação
  • Reset permite limpar dados, muito utilizado para resetar um formulário
  • Submit tem a ação de enviar os dados, pode ser feito com o input também, porém, como button temos mais opções de ações e podemos mudar o nome. Ou seja, temos maior controle sobre suas ações
